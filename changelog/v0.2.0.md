# BookUnbind v0.2.0

Second release, focused on branding (custom app icon end-to-end) and documentation polish.

## тЬи New

### Custom app icon, for real this time

- [`assets/icon.png`](../assets/icon.png) (2048├Ч2048 RGBA) is now the single source of truth for branding.
- **CI-built `.app` / `.exe` / Linux binaries** pick it up automatically via `flet build` and generate the right platform-specific formats (`.icns`, `.ico`, embedded PNG).
- **Dev mode (`python main.py`) on macOS** now also shows the custom icon in the Dock and About dialog. `main.py` clones the Flet desktop runner into a project-local `.flet-local/BookUnbind.app`, converts `icon.png` to a multi-resolution `.icns` with `sips` + `iconutil`, patches `Info.plist` (`CFBundleName`, `CFBundleDisplayName`, `CFBundleShortVersionString`, `NSHumanReadableCopyright`), removes the bundled `Assets.car` and `CFBundleIconName` that used to override the custom icon, bumps mtimes, and calls `lsregister -f` to force Launch Services to refresh its icon cache.
- The shared system-wide Flet runner is never modified тАФ other Flet projects keep their default icon.

### Redesigned README

- Centered hero block with emoji title, subtitle, badges, and a framed icon preview.
- Prominent three-platform "Download" section with large action badges linking straight to `/releases/latest/download/` URLs (so the download link always resolves to the latest version, no README edit needed on future releases).
- Added tech-stack badges for Python, Flet, PyMuPDF and pypdf.

## ЁЯРЫ Fixes

- **Stale `build/macos` shadowed `FLET_VIEW_PATH` in dev mode.** Flet's runner lookup checks `./build/macos/*.app` *before* `FLET_VIEW_PATH`, so an old build left over from a previous `flet build` kept winning over the customized runner. `main.py` now removes `build/macos` on startup when its icon is older than `assets/icon.png`.
- **macOS About dialog / Dock kept showing the Flet icon.** Root cause: `Assets.car` + `CFBundleIconName` together told macOS to ignore `AppIcon.icns`. Both are now removed from the cloned dev runner.

## ЁЯз░ Internals

- Added `.flet-local/` and `assets/icon.icns` to `.gitignore` (both are generated artefacts).
- The `.icns` is regenerated only when `assets/icon.png` changes, so startup stays fast on subsequent dev launches.
