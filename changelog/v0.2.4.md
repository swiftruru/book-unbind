## v0.2.4

### Features
- Windows: ship an Inno Setup installer (`BookUnbind-Setup-<version>.exe`) alongside the portable zip. Installs to `%LocalAppData%\Programs\BookUnbind` by default (per-user, no UAC) and adds Start Menu / optional Desktop shortcuts.

### Why
Running `BookUnbind.exe` directly from a Parallels / VMware shared folder (`C:\Mac\...`) launches a blank window because Flet's embedded Python can't load `app.zip` from those virtual file systems. The installer forces installation onto a real NTFS volume, sidestepping the issue for end users.

### Notes
- The portable zip is still published; power users who copy it onto a local disk can keep using it.
